My Projects

Railway Ticket Booking System

The Railway Ticket Booking System is a simple C program that simulates seat booking using a 1D array. It allows users to book seats, cancel bookings, and view seat status through a menu-driven system. This project helps in understanding arrays, loops, conditions, switch-case, and functions while applying programming concepts to a real-life booking scenario.

Dynamic To-Do List

The Dynamic To-Do List Project is a simple web application that allows users to add and delete tasks interactively. It uses HTML for structure, CSS for styling, and JavaScript with DOM manipulation to update the task list in real time without reloading the page. This project helps in understanding event handling, dynamic content creation, and basic front-end logic, making it a practical example of interactive web development.

Financial Planner Workbook

The Financial Planner Workbook Project is an Excel-based project designed to help users manage and track their personal finances efficiently. It includes sheets for expenses, budgeting, and charts, using tools like data validation, formulas (such as SUMIF), and charts to analyze spending patterns. This project helps in understanding organized data management, financial analysis, and practical use of Excel functions for real-life budgeting.
